SmartPlace

Overview

SmartPlace functions as an AI-powered campus placement ecosystem designed to modernize the traditional recruitment process within an academic environment. It replaces manual, inefficient keyword filtering with advanced Deep Learning, specifically using Sentence-BERT, to create semantically accurate matches between student resumes and job descriptions.

Core Functionalities

Semantic Resume Matching: The system leverages Sentence-BERT to compare student resumes against job descriptions, calculating a percentage match score based on meaning rather than simple keyword matching.
NLP-Based Resume Parsing: An integrated NLP parser automatically extracts skills, projects, and experiences from student-uploaded PDF resumes, eliminating the need for manual data entry.
ATS & Skill Gap Analysis: SmartPlace generates automated ATS scores for candidates and provides a "Missing Skills" report, offering personalized roadmaps for students to enhance their qualifications.
Multi-Role Ecosystem: The platform features distinct dashboards for three user roles:
Students: Access a job match feed, track application/interview status, and receive career advice.
Faculty: View ranked student lists sorted by AI-match scores and filter candidates by specific requirements like CGPA or skills.
Super Admin: Oversee platform analytics, manage credentials, and ensure data privacy within the college environment.

Technical Focus 

This project demonstrates proficiency in full-stack development and AI integration, specifically targeting:
Deep Learning & NLP: Implementation of Transformer models for semantic relevance.
Full-Stack Architecture: Development of a responsive, secure portal with role-based access control.
Data-Driven Decision Making: Transforming unstructured PDF data into ranked, actionable insights for recruitment and upskilling.
